挥洒汗水，拼搏奋进，记录苏中运动健儿们的风采 【一周校园事 第一期】


运动会

#SPORTS

2021年9月18日，苏州中学校内运动会顺利举行，在经过精彩激烈的角逐后，一批优秀的运动健儿被选拔出来参加即将举办的四校联合运动会，响应”文明其精神，野蛮其体魄“的号召，弘扬奥林匹克精神，为校园文化建设添砖加瓦。


运动员们矫健的身姿，有力的步伐，将拼搏的精神传递给每一位学子，将努力的汗水挥洒在每一处赛场

SPORTS











奋力跃起，

如展翅飞翔的海燕，

超越天空；



勇往直前，

健步如飞，

突破速度的极限

接力赛场上，

同学们传递着热情与自信，

为共同的目标奔向终点

欢呼声,鼓舞声在运动场上回荡，

迸发出的，

是青年人的自信与勇敢，

是如朝阳般的勃勃生机。

02

运动会如火如荼的进行着，与此同时，我们特地采访到了几位表现出色的运动员，看看他们怎么说


k1-9 周杨

Q1:同学你好，首先祝贺你在1000米比赛中取得的出色成绩，请问你平时是如何训练的呢？


A1:我会坚持骑行，周末50公里，平时也是自行车上下学。自行车能大大提高腿部力量！自从骑了车，我从3.40一路提升到3.09，这个法宝大家一定要收下哦


Q2:哈哈，真是努力的选手，你的建议我们收下了，请问本次初赛是否有什么觉得有些遗憾的地方呢


A2:暑假里，开学初生了两次大病，训练中断了。又得到1000米要预赛的消息，临时上阵。周三我都无力跑完1000，周四跑了3.56，周五才找回感觉跑进了3.20。3.13确实是我现在的极限了，不过离决赛还有10天，我会尽力挖掘自己的潜能


Q3:好的，请问你为什么会喜欢上长跑呢？


A3：长跑，我觉得锻炼的就是一种毅力。我想，长跑不仅对健康有益，更对人生有益。学会了长跑，学会了节奏，也就学会了生活


C：好的，谢谢你的回答，期待你可以在决赛的赛场上取得自己满意的成绩。






k2-3 曹兆宇

　q1:曹同学你好，首先祝贺你在预选赛中取得的出色成绩，请问你一直以来都有在准备这场比赛吗


  a1：同学你好，唔并不是，只有预选赛前一星期内做过些许恢复性练习，其他时间并没有练习过


  q2:这次比赛当中我们有看到你突发伤病，其实你已经可以拿到决赛的资格了，为什么还会坚持到底呢，当时心里在想些什么


  a2：这个嘛，当时心里其实也没有想很多，萦绕在脑海中的有两句话，第一句是，不能让三班丢脸，第二句是，我就算是要输也要输给对手，而不是输给我自己的放弃。就像海明威先生的一句话“人可以被打倒，但不能被打败”


 q3：同学们祝愿你在决赛中取得优秀成绩的同时也很关心你的身体情况，对此有什么对他们想说的吗


 a3：同学们以及老师们的帮助与关心真的很令我感动，这同样也是支持我拼搏到最后的一大支柱。我记得最后一跳时围聚在赛场旁同学们鼓励的掌声，记得在我身边不断问我还行不行的同学，记得老师和几位同学把我送到医务室时那焦急的目光...两三个小时的赛程中发生了很多无法忘却的美好回忆，我很想再对关心我的同学和老师们说一声“谢谢你们”

还有一个小插曲，到了决前三的那一跳时，蔡老师跑过来问我现在脚伤到底怎么样了，告诉我别再跳了我们弃赛。在医务室的时候，蔡老师望着我告诉我，以后别再做这种事了，声音轻轻的。www超感动


c1：好的，谢谢曹同学的用心分享，希望你可以早日康复并在决赛中取得满意的成绩


k2-2 江昊林

q1:江同学你好，先祝贺你在预选赛中取得的出色成绩，本次的成绩是否达到了你的预期呢


a1:还算满意，但仍有做的不够好的地方，还需努力

q2:好的，请问在准备决赛的这几天你打算做些什么呢

a2:每天适量运动维持状态，同时学习起跑技术，弥补短板。

q3:收到，请问有什么想对支持自己同学说的吗？

a3:非常非常感谢我的朋友们，正是因为他们，我不是一个人在战斗。

c:好的，谢谢你的回答，希望你可以在决赛时取得满意的成绩，加油！



相信同学们的努力定会在即将到来的四校联合运动会中化作为优异的成绩。在此，New Current编辑部全体成员祝愿本次运动会顺利举行，向辛苦筹备的各位老师致以最真挚的感谢。
